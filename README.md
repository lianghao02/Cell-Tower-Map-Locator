# 📱 Cell Phone Locator (手機門號即時定位)

![Version](https://img.shields.io/badge/Version-v2.0-blue)
![License](https://img.shields.io/badge/License-Proprietary-red)

這是一個專為手機門號即時定位設計的專業級輔助工具，能快速解析電信查詢回傳的雜亂文字，自動提取關鍵資訊（座標、時間、門號）並進行地圖定位與導航。**v2.0 版本更引入了強大的目標地址空間關聯分析與大一統控制台。**

---

## ✨ 核心功能

* **⚡ 智慧解析 (Smart Parse)**：
  * 支援直接貼上電信公司回傳的簡訊或查詢結果。
  * 自動辨識 **經緯度座標** (TWD97/WGS84 自動判讀)。
  * 自動提取 **目標門號**、**定位請求時間**、**註冊基地台時間**。
  * 自動判讀 **方位角 (Azimuth)** 並在地圖上繪製扇形區域。

* **🏠 空間關聯與目標對比 (v2.0 全新功能)**：
  * **三合一防禦性定位**：支援「OSM 地址解析」、「手動輸入經緯度」與「地圖十字選點」，並可在地圖上自由拖曳 🏠 標記微調目標房屋位置。
  * **智慧中文模糊容錯**：搜尋前自動進行繁簡轉換、正則過濾樓層房號雜訊，大幅提高偏鄉門牌的解析成功率。
  * **直線距離與覆蓋分析**：自動計算基地台與目標 Pin 之間的直線距離、相對方位角。若有方位角資訊，會智慧判定目標是否落在信號發射扇形內，並以虛線顏色 (綠色🎯/紅色❌) 即時標示。
  * **參數調整實時重算**：在抽屜面板變更發射半徑或夾角時，空間覆蓋判定與虛線顏色會即時變色刷新。

* **🎛️ 大一統折疊式控制台 (v2.0 全新功能)**：
  * **一頁免滾動佈局**：重整網頁版面，將所有輸入欄位與最近查詢歷史卡片合併入左側控制卡片，與右側地圖完美對齊。
  * **流暢折疊動畫**：採用貝氏曲線動畫開合三大面板：`⚡ 基地台定位資訊`、`🏠 目標關連位置`、`📜 最近查詢歷史`，並具備狀態指示箭頭。
  * **智慧視野自動調焦 (Auto Fit Bounds)**：載入含地址的定位時，地圖自動智慧縮放，將兩點完美包覆呈現在螢幕中央。

* **📋 便捷操作**：
  * **一鍵複製**：將整理好的格式化資訊（若有地址將自動修正輸出，附帶直線距離與比對結果）複製到剪貼簿。
  * **快速分享**：支援一鍵傳送至 **LINE** 或 **Telegram** 群組。
  * **歷史紀錄實時同步**：自動儲存最近的查詢紀錄，每當在地圖上拖曳 Pin 或選點微調位置時，空間資訊會實時寫回 LocalStorage，保持歷史追溯一致性。

## 🛠️ 使用教學

1. **貼上資訊**：將電信查詢結果（簡訊或文字檔）貼入上方文字框，或使用「貼上」按鈕。
2. **解析定位**：點擊「⚡ 解析並定位」按鈕，系統將自動提取資訊並展開第一個摺疊面板。
3. **關聯目標位置 (選填)**：
   * 展開 `🏠 目標關連位置`，輸入目標地址點擊 `🔍` 解析定位，或點擊 `🎯 地圖選點` 在地圖上點擊位置。
   * 您可直接在畫面上拖曳橘色 🏠 Pin 到屋頂，左側的「空間關聯分析」面板將即時顯示測距與信號覆蓋狀態。
4. **回報與導航**：
   * 點擊「Google Map」開啟導航。
   * 點擊「複製資訊」取得格式化文字報告。
   * 點擊「LINE」或「Telegram」快速回報。

## 📋 輸出格式說明

本工具會根據**有無目標地址**自動產生對應的格式化報告，方便快速閱讀與分享：

### 格式 A（僅有基地台定位時）：
```text
https://www.google.com/maps?q=23.123456,120.123456
定位門號: 0912345678
定位時間: 2026/07/15 09:30:00
註冊時間: 2026/07/15 09:28:00
定位經緯度: 23.123456, 120.123456 (方位:90)
專用連結: http://... (供點擊開啟專案圖資)
```

### 格式 B（包含目標對比地址時）：
```text
https://www.google.com/maps?q=23.123456,120.123456
定位門號: 0912345678
定位時間: 2026/07/15 09:30:00
註冊時間: 2026/07/15 09:28:00
定位經緯度: 23.123456, 120.123456 (方位:90)

🏠 目標關連位置: 臺南市將軍區南19線3鄰5之1號
📍 目標經緯度: 23.124567, 120.124567
📏 直線距離: 約 185 公尺
🧭 相對方位角: 78° (🎯 位於發射扇形範圍內)
專用連結: http://... (供點擊開啟專案圖資)
```

## 🔧 技術規格

* **前端架構**：原生 HTML5 / CSS3 (Tailwind CSS CDN) / JavaScript (ES6+)。
* **地圖核心**：Leaflet.js (OpenStreetMap)。
* **空間運算**：球面三角學與航向角（Bearing）公式。
* **資安規範**：純前端運行，Local Storage 本地保存，確保公務機敏資料絕不外洩。
