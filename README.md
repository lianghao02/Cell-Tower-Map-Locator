# 📱 Cell Phone Locator (手機門號即時定位)

![Version](https://img.shields.io/badge/Version-v3.0-emerald)
![License](https://img.shields.io/badge/License-Proprietary-red)

這是一個專為手機門號即時定位設計的專業級 GIS 輔助工具，能快速解析電信查詢回傳的雜亂文字，自動提取關鍵資訊（座標、時間、門號）並進行地圖定位與導航。**v3.0 版本迎來了重大革新，全面重構為「全畫面滿版背景地圖 ＋ 左上角懸浮折疊控制台」架換。**

---

## ✨ v3.0 核心升級與功能

* **🗺️ 全畫面滿版背景地圖 (z-0)**：
  * 地圖容器以全螢幕背景呈現，搭配無拉 bar 佈局，帶來極具現代感與沉浸式的地理資訊視覺體驗。
  * 前台覆蓋層使用點擊穿透技術（`pointer-events` 控制），確保非控制台區域可直接拖曳與縮放地圖。

* **🎛️ 左上角懸浮摺疊卡片 (z-20)**：
  * 控制台採用半透明磨砂玻璃卡片設計（`bg-white/85 backdrop-blur-xl`），優雅鋪設於地圖上方。
  * **一鍵向左收合拉手**：卡片右側設有平滑拉手按鈕，點擊可將卡片完整收回至左邊界外（平移 `-376px`），留給地圖 100% 視野。

* **🗂️ 四大 Tabs 分頁資訊分流**：
  * 卡片內部橫向設立 `⚡ 智慧解析`、`📡 定位資料`、`🏠 空間對比`、`📜 最近歷史` 四大標籤分頁。
  * 限制面板縱向高度並支援內部滾動，徹底解決欄位過長撐開視窗的痛點。

* **✈️ 智慧 Pan & Zoom 鏡頭對焦飛行引擎**：
  * 搜尋地址或點擊選點成功時 ➡️ 地圖鏡頭自動飛行平移至該目標位置，並**預設放大至清晰的 18 級**！
  * 簡訊解析定位成功時 ➡️ 地圖鏡頭自動飛行平移至基地台，並放大至 18 級。
  * **手動拖曳防跳功能**：手動拖曳 Pin 針微調座標時，地圖鏡頭自動保持靜止，確保微調流暢。

* **🔍 智慧地標比對與 8 公里超距警告 (防同名誤判)**：
  * **保留原始輸入**：搜尋地址時保留使用者原始打字（如：新化派出所），防止被解析地名默默覆蓋。
  * **字詞吻合度比對**：提取搜尋詞核心關鍵字，比對解析地標名稱（例如：若搜尋新化，解析出中正派出所，因字詞不吻合將自動彈出紅色預警）。
  * **超距偏離預警**：當解析地點與基地台距離大於 **8 公里**時，自動彈出紅色超距警報，並提供「地圖選點」與「搜尋路名拖曳 Pin 針」等智慧操作指引。
  * **座標搜尋豁免**：若使用者直接輸入純數值經緯度搜尋，系統會正則識別並自動豁免字詞比對，防止座標被判定為不吻合。

* **🔎 高清 22 級地圖放大**：
  * 解鎖 Leaflet 原生 18 級縮放限制，最高支援放大至 **22 級**。
  * 啟用 `maxNativeZoom: 19` 智慧插值放大，防範無瓦片圖資時的破圖灰格現象，輕鬆看清街道門牌與角度細節。

## 🛠️ 使用教學

1. **貼上資訊**：將電信查詢結果（簡訊或文字檔）貼入上方文字框，或使用「貼上」按鈕。
2. **解析定位**：點擊「⚡ 解析並定位」按鈕，系統將自動提取資訊並展開第一個分頁。
3. **關聯目標位置 (選填)**：
   * 切換至 `🏠 空間對比` 分頁，輸入目標地址點擊 `🔍` 解析定位，或點擊 `🎯 地圖選點` 在地圖上點擊位置。
   * 您可直接在畫面上拖曳橘色 🏠 Pin 到屋頂，左側的「空間關聯分析」面板將即時顯示測距與信號覆蓋狀態。
4. **回報與導航**：
   * 點擊「Google Map」開啟導航。
   * 點擊「複製資訊」取得格式化文字報告。
   * 點擊「LINE」或「Telegram」快速回報。

本工具會根據**有無目標地址**自動產生對應的格式化報告，方便快速閱讀與分享：

### 格式 A（僅有基地台定位時）：
```text
https://www.google.com/maps?q=23.123456,120.123456
定位門號: 0912345678
定位時間: 2026/07/15 09:30:00
註冊時間: 2026/07/15 09:28:00
定位經緯度: 23.123456, 120.123456 (方位:90)
專用連結: http://... (供點擊開啟專案圖資)
```

### 格式 B（包含目標對比地址時）：
```text
https://www.google.com/maps?q=23.123456,120.123456

註冊時間: 2026/07/15 09:28:00
定位經緯度: 23.123456, 120.123456 (方位:90)

🏠 目標關連位置: 臺南市將軍區南19線3鄰5之1號
📍 目標經緯度: 23.124567, 120.124567
📏 直線距離: 約 185 公尺
🧭 相對方位角: 78° (🎯 位於發射扇形範圍內)
專用連結: http://... (供點擊開啟專案圖資)
```

## 🔧 技術規格

* **前端架構**：原生 HTML5 / CSS3 (Tailwind CSS CDN) / JavaScript (ES6+)。
* **地圖核心**：Leaflet.js (OpenStreetMap)。
* **空間運算**：球面三角學與航向角（Bearing）公式。
* **資安規範**：純前端運行，Local Storage 本地保存，確保公務機敏資料絕不外洩。

---

## ⚠️ 注意事項

* 本工具僅供公務使用，請遵守相關資安與保密協定。
* 座標與地址解析存在少許誤差，實際位置請以現場狀況為準。一鍵傳送至 **LINE** 或 **Telegram** 群組。
  * **歷史紀錄實時同步**：自動儲存最近的查詢紀錄，每當在地圖上拖曳 Pin 或選點微調位置時，空間資訊會實時寫回 LocalStorage，保持歷史追溯一致性。
